# Images and Media

This directory contains images and media files related to Govardhanapuram Gosala.

## Purpose

- Showcase the gosala facilities
- Document cow care activities
- Share success stories
- Promote community engagement

## Usage Guidelines

When adding images to this directory:

1. Use descriptive filenames (e.g., `cow-shelter-2025.jpg`)
2. Optimize images for web use
3. Include attribution if required
4. Ensure images respect privacy and dignity

## Image Categories

- **Facilities**: Photos of the gosala infrastructure
- **Animals**: Pictures of cows under care
- **Activities**: Community events and volunteer activities
- **Products**: Cow-based products (milk, ghee, compost, etc.)

---

Please ensure all images are appropriate and align with the gosala's mission and values.
